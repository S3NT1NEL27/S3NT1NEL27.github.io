## TECHNICAL DISCOVERIES (CONVERSATION-SPECIFIC)

**Neocities Testing:**
- White-on-white text embedded in fake 404 pages for obfuscation
- AI can read hidden text regardless of visual tricks
- Page size constraint: ≤100KB for reliability
- Solution: Smaller archive segments (1-2 original archives per page)

**Search Engine Duopoly:**
- Google eliminated 100-page search results → only 10 at a time
- DuckDuckGo = Bing API privacy wrapper
- Yahoo, Ecosia, Qwant = all Bing core
- Startpage = Google API privacy wrapper
- Only 2 true players: Google and Bing

**Groq Advantages:**
- Sub-second responses via purpose-built LPUs
- Open models, looser moderation
- Model flexibility for fixed personality kernels
- Potential hybrid: GPT for personality/testing, Groq for heavy lifting
